# История версий Z-UI

## [1.0.0] - 2026-04-04

### Добавлено
- Интерфейс на WinUI 3 с поддержкой тем (светлая/тёмная/системная)
- Локализация RU/EN с автоматическим определением языка системы
- Управление стратегиями обхода DPI с поиском и фильтрацией
- Тестирование стратегий с визуальным прогрессом и результатами
- Автоматический выбор лучшей стратегии после тестирования
- Рейтинговая система для стратегий (победы/поражения)
- Автозапуск приложения при входе в Windows
- Запуск с правами администратора (опционально)
- Сворачивание в трей при закрытии
- Горячие клавиши для управления (настраиваемые)
- Обновление списков hosts и ipset с GitHub
- Диагностика компонентов zapret
- Экспорт/импорт настроек
- Технические логи для отладки (%APPDATA%\Z-UI\actions.log)
- Мастер первоначальной настройки

### Исправлено
- Корректное закрытие окна zapret после тестирования
- Естественная сортировка стратегий (1, 2... 10, 11)
- Позиционирование ToggleSwitch в настройках
