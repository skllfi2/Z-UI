# Анализ структуры проекта Z-UI

## Обзор общей структуры

### Файлы документации:
- **README.md** - основной файл проекта с описанием функциональности, техническим стеком, системными требованиями и примерами использования
- **CHANGELOG.md** - история версий с описаниемм изменений, добавлений и исправлений
- **CONTRIBUTING.md** - руководство для контрибьюторов с описанием рабочего процесса, требованиями к коду и гайдами по созданию PR

### Структура документации:
- **docs/** - директория с подробной документацией:
  - `architecture/` - архитектурные решения
  - `deployment/` - развёртывание и установка
  - `development/` - руководство по разработке
  - `troubleshooting/` - решение проблем
  - `user-guide/` - инструкция пользователя

### Проектные файлы:
- **Z-UI.slnx** - решение Visual Studio
- **Z-UI/** - основной проект (.NET 10.0, WinUI)
- **Z-UI.csproj** - файл проекта с зависимостями
- **Assets/** - иконки приложения
- **zapret/** - бинарные файлы и скрипты для работы с запретными списками:
  - `winws/` - бинарные WinDivert
  - `lists/` - текстовые списки запрещенных ресурсов
  - `strategies/` - скрипты стратегий обхода

## Технологический стек
- **.NET 10.0** - текущая версия фреймворка .NET
- **WinUI 3** - современный UI фреймворк от Microsoft
- **Windows 10/11** - целевая платформа
- **C# 12** - язык разработки

## Архитектура
- **MVVM** - Model-View-ViewModel паттерн для разделения бизнес-логики от UI
- **WinUI 3** - современный UI фреймворк
- **WinDivert** - системная фильтрация сетевого трафика

## Паттерны документации
- **Semantic Versioning** - версионирование по SemVer
- **Conventional Commits** - стандартные сообщения коммитов
- **Markdown** - все документы в формате разметки
- **Git workflow** - стандартный рабочий процесс с фичирными ветками

## Особенности проекта
- Фокус на запрещении информации в России
- Работа с системной фильтрацией трафика
- Мультиязычный интерфейс на русском и английском языках
- Поддержка Windows 10/11 самых последних версий

```
Z-UI/

├── CHANGELOG.md                 # История версий
├── CONTRIBUTING.md              # Руководство для контрибьюторов
├── docs/                        # Детальная документация
│   ├── architecture/           # Архитектурные решения
│   ├── deployment/             # Развёртывание
│   ├── development/            # Разработка
│   ├── troubleshooting/        # Поиск и исправление ошибок
│   └── user-guide/             # Инструкция пользователя
├── Z-UI/                        # Основной проект
├── Z-UI.slnx                   # Решение Visual Studio
├── Assets/                      # Иконки приложения
└── zapret/                      # Бинарные файлы и скрипты
    ├── winws/                  # Бинарные WinDivert
    ├── lists/                  # Списки запрещенных ресурсов

```
